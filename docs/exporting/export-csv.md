## Export To CSV

You can easily export reports as a CSV file. Simply select `-ExportTo CSV` as shown below:

``` PowerShell
$p = @{
    Instance = 'Azure';
    Collect = 'All';
    PromptBehavior = 'SelectAccount';
    AllSubscriptions = $true;
    TenantID = '00000000-0000-0000-0000-000000000000';
    ExportTo = 'CSV';
}
Invoke-Monkey365 @p
```
